# Guess the Number
 It's a simple console game where you choose the number and Computer try to guess it :)
 You choose how many guess tries Computer has.

------------------------------
 Variables:
    try_num - counter for amount of guesses
    try_max - The maximum amount of guesses
    max - maximum number from which the computer is guessing  
    min - minimum number from which the computer is guessing
    guess - if computer win the game
    ans - variable to store answer if guessed number is higher or lower
    num -  Currently computer guess

Created this to learn basic syntax for Python so It's not complicated
